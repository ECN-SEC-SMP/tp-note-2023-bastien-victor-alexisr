\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classTile}{Tile}} class represents a single tile on the game board. 

Since the game board is a 16x16 grid, there are 256 tiles in total. Each tile can have walls on one or more of its sides, a target and/or a robot.
